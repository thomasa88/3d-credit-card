Use the customizer in OpenSCAD to set card details and export each colored part as STL one by one (F6, F7).

Make sure to tweak the card thickness to e.g. 1.8 mm if you want a more durable toy rather than a realistic card.

Import all STL files at the same time into PrusaSlicer.
It will ask if they should be imported as on multi-material piece if it has been set up to have mulitiple extruders.

It is possible to pretend that a single tool printer has multiple extruders by setting the tool changing code to manual color change (`M600`) and disabling the wipe tower.
